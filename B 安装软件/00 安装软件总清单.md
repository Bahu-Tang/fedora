## 0.1 📋 从 Windows 到 Fedora：软件安装对照手册

### 0.1.1 一、基本原则（先记住这三条）

| 软件类型           | 推荐安装方式                    | 为什么          |
| -------------- | ------------------------- | ------------ |
| **系统工具/命令行软件** | `sudo dnf install`        | 与系统集成好，更新方便  |
| **桌面图形应用**     | `flatpak install flathub` | 沙箱隔离，跨发行版兼容  |

### 0.1.3 三、特殊软件说明

#### 0.1.3.1 🐳 Docker（重要）
```bash
# 1. 安装 Docker
sudo dnf install docker docker-compose

# 2. 启动服务并设置开机自启
sudo systemctl enable --now docker

# 3. 把自己加入 docker 组（免 sudo）
sudo usermod -aG docker $USER

# 4. 重新登录生效
```
**关键**：Docker 默认数据在 `/var/lib/docker`，如果你担心空间，可以用之前的方法迁移到 `/home`。

#### 0.1.3.2 🐍 Conda（重要）
```bash
# 下载 Miniconda 安装脚本
wget https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh

# 指定安装到你规划的目录
bash Miniconda3-latest-Linux-x86_64.sh -p ~/软件/Computer/miniconda3

# 安装完成后，添加到 PATH
echo 'export PATH="$HOME/软件/Computer/miniconda3/bin:$PATH"' >> ~/.bashrc
source ~/.bashrc
```

#### 0.1.3.3 💻 VS Code
**推荐方式**：DNF 安装（最省心）
```bash
sudo dnf install code
```
**或者**：官网下载 `.rpm` 包手动安装。

### 0.1.4 四、快速安装清单（一键复制）

把这些命令按顺序执行，就能装完大部分常用软件：

```bash
# ===== 系统工具 (DNF) =====
sudo dnf install git nodejs docker docker-compose wireshark gcc make cmake unrar -y

# ===== Flatpak 应用 =====
flatpak install flathub md.obsidian.Obsidian com.obsproject.Studio org.blender.Blender com.discordapp.Discord com.google.Chrome org.mozilla.firefox org.videolan.VLC org.gimp.GIMP com.qq.QQ com.tencent.WeChat com.wps.Office com.tencent.wemeet -y

# ===== VS Code =====
sudo dnf install code -y

# ===== Docker 启动 =====
sudo systemctl enable --now docker
sudo usermod -aG docker $USER
```
